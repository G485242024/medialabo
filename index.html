各授業の学習事項

第1回 WWW と HTML

第一回目の授業では、WWWを構成する技術要素と、HTMLの要素と構成について学びました。WWWについてはWWWページのURL、通信、Webページの部品について学びました。また、HTMLは細かい要素を使って構成していることがわかりました。WebページとHTMLは、どちらもパズルのようになっていて似ているように感じました。

演習の成果: 演習1-4:HTMLのさまざまな要素

第2回 CSS による Webページの装飾

第2回目の授業では、CSSによるWebページのの装飾を学びました。CSSとは要素の見た目を変えるもので、前回の授業で学んだHTMLの後にCSSによって装飾を施すものです。CSSは「.css」が末尾につくファイルから装飾をつけるためのプログラミングをします。CSSによって付けられる装飾は「色」のほかに「フォントサイズ、フォントタイプ」、背景の色などさまざまなものがありました。

演習の成果: 演習2-3:Webブラウザ説明ページの装飾

第3回 JavaScript の基本

第3回目の授業では、JavaScriptで出来ることと、コンソールの使い方について学びました。JavaScriptとは、Webページ上の複雑な機能を実現するプログラミング言語です。今回の授業では出力先を「コンソール」にしました。コンソールを活用して、JavaSriptの追加をしました。また、授業内でわざとエラーを発生させて、デバッグに慣れることもしました。
